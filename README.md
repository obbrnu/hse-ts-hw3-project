# Моделирование сезонности в временных рядах

Исследовательский проект по анализу эффективности различных подходов к моделированию сезонности.

## Постановка задачи

**Гипотеза:** Как лучше всего моделировать сезонность в временных рядах?

**Исследуются:**
- Эффективность различных наборов признаков (лаги, сезонные лаги, календарные, Фурье)
- Сравнение ML-модели (CatBoost) с baseline методами (Naive, SeasonalNaive, AutoTheta, AutoETS)
- Зависимость качества от горизонта прогнозирования
- Зависимость качества от силы сезонности

**Данные:** 50 временных рядов из M4 Competition

**Протокол:** Time Series Cross-Validation (3 фолда)

## Установка и запуск

```bash
# Установка зависимостей
pip install -r requirements.txt

# Запуск эксперимента
python run_experiment.py
```
